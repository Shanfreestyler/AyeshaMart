# QurbaniHub

Bangladesh livestock marketplace for Qurbani animals.

## Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- React
- App Router

## Rules
- Reuse existing components.
- Do not create duplicate UI.
- Mobile-first.
- TypeScript strict mode.
- Tailwind only.
- Keep code simple.
- Never modify auth unless requested.
- Never modify env files.
- Never remove existing features.

## Workflow
Before coding:
1. Search existing components.
2. Reuse code whenever possible.

After coding:
1. Run lint.
2. Run build.
3. Fix TypeScript errors.
4. Report changes.

## Priorities
1. Seller Dashboard
2. Admin Dashboard
3. Listing Management
4. Search & Filters
5. SEO
