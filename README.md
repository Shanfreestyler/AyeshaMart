# QurbaniHub

**Alpha MVP** – Premium Online Cattle Marketplace for Bangladesh

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion (moderate animations)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure
- `/src/app` – Pages (Homepage, Browse Cattle, Cattle Details, About Us)
- `/src/components` – UI & shared components
- `/src/lib/mockData` – Mock cattle data
- `/src/types` – TypeScript interfaces

## Design
- Cream background (#FFFDF9) with dark-green accents (#14532d)
- Focus on local Bangladeshi cattle breeds
- Premium, clean UI for concept validation
